### [A2SIT — Community Prioritization & Vulnerability Analysis](./a2sit/README.md)

A2SIT (Area-based Analysis Supporting Identification Tool) is an open-source, data-driven web application developed for UNHCR Guatemala that enables humanitarian actors to build composite vulnerability indices and prioritize communities in need of protection interventions. Built in R with the Shiny framework and the COINr statistical package, the tool transforms multi-dimensional socioeconomic and mobility data into interactive maps and sub-national rankings. It applies Data Science and GIS methods to support transparent, evidence-based decision-making across humanitarian operations.

### [AI for Call Center Conversational Data Analysis](./conversational_data_analysis/README.md)

This graduation project for the Master's in Data Science at PUC Chile designs and implements an end-to-end AI pipeline to process, analyze, and visualize call center conversational data — from raw audio to an interactive Power BI dashboard. The system combines open-source tools (OpenAI Whisper, Microsoft Presidio, pyannote) with LLM-based structuring (GPT-4o-mini) and a Random Forest classifier that achieved 98.6% cross-validation accuracy, complemented by SHAP interpretability. Privacy protection is a core design principle: transcription, anonymization, and all local processing run within the organization's own infrastructure.

### [LLM for Transcription & Analysis of Participatory Diagnostics with Refugees](./llm_transcription_refugees/README.md)

This project applies Large Language Models to UNHCR Guatemala's participatory assessment methodology, converting field audio recordings of focus group discussions with refugees and displaced populations into anonymized, structured, query-ready text. Using OpenAI Whisper for Spanish-language transcription and a private Azure OpenAI instance for LLM-powered qualitative analysis, the pipeline enables protection staff to extract cross-cutting insights across approximately 50 nationally conducted focus groups. The solution systematizes qualitative humanitarian data at scale while preserving participant confidentiality through a locally run anonymization stage.
